# TrayCppExample
Cpp Tray 기능 예제입니다.

### 🔍 구현 기능
1. 대화상자의 x를 누르면 창이 사라지며 트레이에 등록된다. 
2. 등록된 아이콘에 커서를 올렸을 때 툴팁이 나온다.
3. 우클릭 시 메뉴가 뜨고 Open 버튼을 눌렀을 때 다이얼로그가 열린다. Close 버튼을 클릭하면 다이얼로그가 종료된다.
